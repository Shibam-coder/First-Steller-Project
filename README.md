# First-Steller-Project
# Crowdsourcing Smart Contract (Soroban)

## Project Description
This is a basic crowdsourcing smart contract built using Soroban on the Stellar blockchain.

It allows users to create projects and fund them in a simple way.

---

## What it does

- Create a project with an ID and goal
- Add funds to a project
- Check total funds of a project

---

## Features

- Simple and beginner-friendly
- Stores project data on blockchain
- Anyone can fund a project
- Easy to understand and extend
